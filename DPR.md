# Detailed Project Report (DPR)

# Panchayat Beneficiary Management & Decision Support System

## Executive Summary
The project develops a centralized beneficiary management platform for rural Panchayats. It helps identify, organize and prioritize elderly citizens and persons with disabilities for welfare schemes and health services.

## Problem Statement
Beneficiary information is fragmented, outdated and manually maintained, leading to delays, repeated surveys and missed welfare opportunities.

## Objectives
- Centralize data
- Improve planning
- Prioritize vulnerable citizens
- Reduce duplicate surveys
- Generate analytics
- Improve transparency

## Stakeholders
Primary:
- Elderly
- Persons with disabilities

Secondary:
- Panchayat officials
- ASHA workers
- Anganwadi workers
- Ward members

## Pilot
Location: One Panchayat
Coverage: 2 wards
Beneficiaries: 100–300
Duration: 4 weeks

## Functional Requirements
FR1 Login
FR2 Register/Edit beneficiary
FR3 Upload documents
FR4 Track requests
FR5 Search/filter
FR6 Dashboard
FR7 Reports
FR8 Welfare events

## Workflow
Field survey
→ Verification
→ Digital profile
→ Analytics
→ Priority generation
→ Welfare planning
→ Service delivery
→ Status update

## Non-functional Requirements
- Mobile responsive
- Fast search
- Secure authentication
- Simple UI
- Large fonts

## Success Metrics
- Registration <5 min
- Search <2 sec
- Dashboard <3 sec
- >80% user satisfaction
- Reduced manual effort

## Risks
- Outdated data
- Low adoption
- Limited internet
- Training needs

Mitigation:
Assign update ownership to ASHA/Ward members and periodic verification.
